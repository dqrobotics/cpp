Unit Dual Quaternion constructor.

Returns a DQ object with unitary norm defined as DQ h = r + DQ::E(r)*0.5*p*r. Where, r is a rotation quatrnion composed by a rotation angle and rotation axis. and p is a translation quatrnion composed by three displacements on the three coordinates x,y and z. To create a DQ object using this, type: 'unit_DQ_h = DQ::unitDQ(PI/4, 0,0,1, 2.2,0,2.457);' for example.

Parameters
rot_angleis the angle rotation of the unit DQ.
x_axis,y_axisand z_axis are the axis which are rotated
x_trans,y_transand z_trans are the displacements of translation on the respective axis x, y and/or z.
Returns
A DQ object.
